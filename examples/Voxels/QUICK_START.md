## Controls

| Control | Action |
|---------|--------|
| ESC | Open/close settings |
| Mouse move | Hover to preview |
| Preview Face slider | Choose face (0-5) |
| Regenerate Maze | New pattern |
| Rows/Columns/Seed | Customize maze |

## Face Numbers

```
0 = Left side   (-X)
1 = Right side  (+X)
2 = Back        (-Z)
3 = Front       (+Z)
4 = Bottom      (-Y)
5 = Top         (+Y)
```
